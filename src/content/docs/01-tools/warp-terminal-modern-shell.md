## 1. 简介

Warp 是一款重新定义了终端体验的工具。它不仅运行速度极快（GPU 加速渲染），还通过创新的 UI 设计让命令行操作变得像在 IDE 中一样直观。

## 2. 核心特性

- **块状交互**：命令与输出被组织成独立的“块”，方便复用命令和分享输出。
- **Warp AI**：内置大模型支持，可以直接用自然语言查询命令或修复错误。
- **协作共享**：支持创建团队共享的命令库（Workflows），方便团队知识沉淀。
- **极速性能**：基于 Rust 开发，即使处理海量日志输出也能保持流畅。
