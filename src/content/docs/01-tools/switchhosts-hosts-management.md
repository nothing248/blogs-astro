## 1. 简介

SwitchHosts 是一个用于管理和快速切换 hosts 文件的开源应用。它提供了直观的界面，允许用户创建多个 hosts 配置，并支持根据需求进行组合或切换。

## 2. 核心特性

- **多方案管理**：可以为不同的项目设置独立的 hosts。
- **远程订阅**：支持自动从远程 URL 获取并更新 hosts 内容。
- **系统菜单集成**：支持从系统菜单栏快速切换。
