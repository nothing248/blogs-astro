## 1. 简介

**Disqus** 是一个非常流行的第三方评论系统。它采用“评论即服务”的理念，让个人站长或开发者无需开发复杂的后端评论逻辑，只需在前端嵌入一段 JavaScript 代码，即可获得一个功能完备、支持社交账号登录且具备防垃圾信息的评论板块。

## 2. 核心功能

- **多平台登录**：支持 Google, Facebook, Twitter 及 Disqus 原生账号登录。
- **实时互动**：支持评论嵌套、点赞、关注等社交功能。
- **管理后台**：提供强大的评论审核、用户屏蔽和垃圾评论过滤工具。
