## 1. 简介

**Eolink (Eolinker)** 是国内领先的 API 全生命周期管理平台。它集成了 API 文档管理、API 研发管理、API 自动化测试、API 网关及监控等功能，帮助企业构建标准化的 API 工作流。

## 2. 核心价值

- **文档自动化**：支持代码注释自动生成文档，确保代码与文档同步。
- **零代码测试**：提供可视化的测试用例编排，降低接口测试门槛。
- **团队协作**：支持多成员权限管理，方便前后端及测试人员的高效配合。
