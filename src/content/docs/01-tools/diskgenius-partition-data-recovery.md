## 1. 简介

**DiskGenius**（曾用名 DiskMan）是 Windows 平台上历史悠久且功能强大的 **全能型硬盘工具**。它不仅是装机维护的必备神器，更是数据丢失时的首选救援工具。

---

## 2. 核心功能板块

### 2.1. 数据恢复 (核心优势)

- **误删除/清空回收站恢复**：快速扫描并找回文件。
- **格式化恢复**：对已格式化的分区进行底层深度扫描。
- **分区丢失恢复**：重建分区表，一键找回意外消失的整个盘符。
- **RAID 恢复**：支持虚拟重组 RAID 阵列，解决服务器级数据灾难。

### 2.2. 分区管理

- **无损调整大小**：在不损坏数据的前提下，动态扩大或缩小分区。
- **快速分区**：适合新硬盘，一键完成 GPT/MBR 分区布局。
- **4K 对齐检测**：优化 SSD 性能的关键指标。
- **格式转换**：支持 MBR 与 GPT 互转，动态磁盘转基本磁盘。

### 2.3. 备份与迁移

- **系统迁移**：将当前 Windows 环境完美克隆至新 SSD，免除重装系统的烦恼。
- **分区/硬盘镜像**：类似 Ghost 的功能，支持文件级和扇区级的全盘备份。

### 2.4. 磁盘体检与安全

- **坏道检测与修复**：检查物理磁盘的健康状况。
- **S.M.A.R.T. 信息**：实时监控硬盘温度、通电时长等参数。
- **文件粉碎**：强制抹除敏感数据，使其无法被任何技术手段找回。

## 4. 常见使用场景

> [!important] 重要提示
> 如果需要进行数据恢复，请 **务必将 DiskGenius 安装在非丢失数据的分区** 中，避免对原始数据造成二次覆盖损坏。

- **新购 SSD**：使用“快速分区”并勾选“对齐分区到扇区的整数倍”以开启 4K 对齐。
- **意外格式化**：立即停止对该盘的一切写入，使用“恢复丢失的文件”进行扫描。
