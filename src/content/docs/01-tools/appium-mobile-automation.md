## 1. 简介

**Appium** 是一个开源的自动化测试框架，专门用于移动端原生应用、移动 Web 应用和混合应用的自动化测试。它支持多种操作系统（如 iOS, Android, Windows）和多种编程语言（如 Java, Python, JavaScript 等），其核心优势在于无需重新编译应用即可进行测试。
