## 1. 简介

**Auto.js** 是一款强大的 Android 平台脚本工具，其最大的特点是利用 Android 系统的辅助功能（Accessibility Service）来实现自动化操作，从而在大多数场景下 **不需要 Root 权限**。

开发者可以使用 JavaScript 编写脚本，实现诸如：

- 自动点击、滑动、长按。
- 基于控件属性（ID, 文本, 类名等）定位元素。
- 图像识别与 OCR 辅助定位。
- 监听通知栏、剪贴板等系统事件。
