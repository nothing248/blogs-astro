## 1. 简介

**微信订阅号** 是微信公众平台的一种账号类型，主要定位于为媒体和个人提供一种新的信息传播方式。与服务号不同，订阅号的核心侧重点在于 **内容推送** 与用户运营，消息会折叠在微信的“订阅号消息”盒子中。

## 2. 内容创作与排版规则

### 2.1. 排版限制与转换工具

订阅号后台编辑器默认采用富文本形式，**不支持原生的 Markdown 语法**。

- **推荐解决方案**：使用 Markdown 在本地撰写后，通过在线排版工具转换并复制到后台。
- **推荐工具**：[Doocs Markdown 排版工具](https://doocs.github.io/md/)

### 2.2. 外部超链接限制

为了防止流量外流和恶意链接传播，微信订阅号文章中 **不支持添加微信生态外的任意超链接**（个人号无法添加外链，已认证的企业主体订阅号可在部分位置配置超链接，但审查严格）。

---

## 3. 核心功能与后台管理

### 3.1. 文章发布机制

- **草稿箱**：日常创作和修改文章的区域。
- **发表记录**：展示历史已发表及已群发的内容。
- **原创保护**：发表文章时可声明原创，开启后系统会自动进行全网文本比对查重。

### 3.2. 交互运营功能

- **自定义菜单**：支持在底部配置菜单栏，菜单项可跳转至文本消息、小程序或者历史公众号文章。
- **赞赏功能**：
  - 作者赞赏资金流向需要绑定“赞赏 IP 账号”。
  - 一个公众号 **最多只能关联 3 个** 赞赏号。
  - 支持配置赞赏成功后的自动回复。
- **自动回复体系**：
  - 关键词回复（当用户输入匹配特定字词时触发）。
  - 收到消息回复（用户发送任意未匹配词信息时的默认答复）。
  - 被关注回复（新用户关注时的首次欢迎词）。
- **其他互动**：支持号内搜索、读者投票、后台私信交流、视频群发弹幕机制等。

---

## 4. 商业化变现（流量主）

- **开通门槛**：公众号需要积累 **100 个粉丝** 后方可申请开通流量主服务。
- **收益方式**：开通后可在文章底部、中部或视频消息中插入微信官方广告，按照点击或曝光度获取分成。

---

## 5. API 开发配置

如需对接外部系统或开发自动聊天机器人，可在“设置与开发 -> 基本配置 -> 服务器配置”中开启开发者模式：

```text
URL: 开发者服务器接收微信消息的接口地址 (如 https://example.com/wechat)
Token: 开发者与微信约定的安全校验字符串
EncodingAESKey: 消息加密密钥
```

> [!WARNING]
> 开启“服务器配置”接管 API 后，公众号后台配置的“自动回复”和“自定义菜单”等可视化配置将会失效，所有交互必须编写后台代码通过微信开放的 XML/JSON 报文格式进行解析与回复。

## 6. 资质与认证限制

- **个人主体订阅号**：**不支持** 进行微信官方的资质认证（蓝 V 标识），且缺少部分高级 API 接口权限。
- **企业主体订阅号**：可进行认证，每年需支付 300 元审核费用，可解锁部分限制。
