## 1. 常见的 LLM 平台

本页整理了常用的大语言模型（LLM）服务提供商及其 API Key 管理后台的快捷访问链接，便于开发者快速获取凭证和查看官方开发文档。

### 1.1. OpenAI

全球领先的闭源大模型提供商，提供 GPT-4o、GPT-4 等业界标杆模型的 API 服务。

- **API 密钥申请入口**：[OpenAI Platform API Keys](https://platform.openai.com/api-keys)

### 1.2. Gemini

Google 推出的多模态大模型，以超长上下文窗口（Context Window）为特色。

> [!note] 资费说明
> 针对部分国家和地区，Gemini API 的基础免费额度（限速）可以直接使用。

- **API 密钥申请入口**：[Google AI Studio](https://aistudio.google.com/app/apikey)

### 1.3. DeepSeek

国内自主研发的高性价比大语言模型，其推理模型（如 DeepSeek-R1）在数学、代码与逻辑推理上表现优异，API 价格极具竞争力。

- **API 密钥申请入口**：[DeepSeek Platform API Keys](https://platform.deepseek.com/api_keys)

### 1.4. 智谱 AI

国内领先的 AI 独角兽企业，提供 GLM 系列模型（如 GLM-4），在中文语境理解与智能 Agent 构建上性能优异。

- **API 密钥申请入口**：[智谱 AI 开放平台](https://open.bigmodel.cn/usercenter/apikeys)

### 1.5. Groq

以极速 LPU（Language Processing Unit）推理芯片闻名，能够为主流开源模型（如 Llama 3、Mixtral）提供每秒数百 Token 的超高速推理 API。

- **API 密钥申请入口**：[Groq Console](https://console.groq.com/keys)

### 1.6. 硅基流动 (SiliconFlow)

国内专业的大模型托管与聚合 API 服务商，整合了 Qwen、DeepSeek、Llama 等多款国内外顶尖开源大模型，提供统一且高可用的 API 端点。

> [!tip] 优惠活动
> 新注册用户通常会获赠一定额度的免费测试消费额度（例如 16 元人民币）。

- **API 密钥申请入口**：[硅基流动控制台](https://cloud.siliconflow.cn/account/)
