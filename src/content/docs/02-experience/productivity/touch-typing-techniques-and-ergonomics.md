## 1. 基础定位：寻找“盲打坐标”

想要提高打字速度并减少手部疲劳，最核心的原则是建立 **“标准指法”** 与保持 **“放松姿态”**。

在开始打字前，手指应自然地落在键盘的 **主操作列 (Home Row)** 上：

- **左手定位：** 食指放在 **F** 键上，中指、无名指、小指分别对应 **D、S、A**。
- **右手定位：** 食指放在 **J** 键上，中指、无名指、小指分别对应 **K、L、;**。
- **大拇指：** 左右手的大拇指都自然悬停在 **空格键 (Space)** 上。

> [!tip]
> **定位盲区技巧：**
> 键盘上的 **F** 和 **J** 键表面通常设计有凸起的小横线或圆点。这是盲打的黄金坐标，只需凭指尖的触感定位即可，无需低头看键盘。

## 2. 十指分工空间逻辑

每个手指在键盘上都有各自的专有“领地”，核心原则是 **就近移动**，击键后手指迅速返回主操作列。

| 手指 | 负责按键范围 |
| :--- | :--- |
| **左手小指** | `Q`, `A`, `Z`, `Shift`, `Ctrl`, `Tab`, `CapsLock` |
| **左手无名指** | `W`, `S`, `X` |
| **左手中指** | `E`, `D`, `C` |
| **左手食指** | `R`, `T`, `F`, `G`, `V`, `B` |
| **左右大拇指** | 专门负责 **Space (空格键)** |
| **右手食指** | `Y`, `U`, `H`, `J`, `N`, `M` |
| **右手中指** | `I`, `K`, `,` (逗号) |
| **右手无名指** | `O`, `L`, `.` (句号) |
| **右手小指** | `P`, `;`, `/`, `Enter`, `Backspace`, `Shift` |

## 3. 正确手型与人体工学姿势

打字是全身的协调动作，不当的姿势极易引发腱鞘炎或肌肉酸痛。正确的工效学规范如下：

- **弓起手背（抓握手型）：** 掌心不要塌陷或贴在键盘上。手指应当呈自然弯曲状（如同 **轻握着一个橘子**），用指尖垂直点击按键，而不是用指腹拍打。
- **手腕平直与悬空：** 手腕避免过度折起或用力向下压在桌沿上。如手腕易疲劳，建议使用键盘掌托，但打字过程中手腕应尽量保持平直。
- **双臂夹角：** 调整座椅高度，使手肘在打字时保持在 **90 度到 110 度** 的自然弯曲夹角，双肩自然下垂放松。
- **轻触即发（无声击键）：** 现代薄膜或机械键盘均有特定的触发键程，轻轻按下触发即可，避免重重敲击以保护指关节。

## 4. 科学打字练习建议

> [!important]
> **1. 坚决杜绝“二指禅”**
> 即使起步阶段打字速度极慢，也要强迫十个手指各司其职。一旦错误的指法形成顽固的肌肉记忆，后期纠正的代价将极其高昂。

> [!note]
> **2. 强迫视线脱离键盘**
> 打字时视线应当锁定屏幕。如果一时记不清按键位置，可在屏幕一侧放置指法分工图进行辅助记忆，严禁低头看手。

> [!tip]
> **3. 节奏与准确率优先**
> 练习初期应将 100% 的准确率作为唯一指标，不要盲目求快。打字应保持像弹钢琴一样的均匀节奏感，稳定的节奏比短时间的爆发快打更有利于提升速度极限。
