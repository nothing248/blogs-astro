## 1. 简介

`gtag.js` 是一个 JavaScript 标记框架和 API，可让你将事件数据发送到 Google Analytics、Google Ads 和 Google Marketing Platform。

## 2. 核心作用

- **统一代码**：只需在网页中添加一次 `gtag.js` 代码，即可向多个 Google 产品发送数据。
- **事件跟踪**：通过简单的 API 调用上报用户交互（如点击、购买）。
