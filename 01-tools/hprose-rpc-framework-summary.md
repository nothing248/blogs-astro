## 1. 简介

Hprose 是一款先进的跨语言 RPC（远程过程调用）框架。它不仅支持多种编程语言（如 Java, Python, Go, Node.js 等），还以其极致的序列化性能和简单易用的 API 著称。

## 2. 核心特性

- **跨语言**：支持 20 多种主流编程语言。
- **高性能**：自研的高效序列化协议。
- **多协议支持**：支持 HTTP, TCP, WebSocket 等多种传输协议。
