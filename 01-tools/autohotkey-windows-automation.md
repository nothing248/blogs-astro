## 1. 简介

**AutoHotkey (AHK)** 是一款功能强大的开源脚本语言，专门为 Windows 用户设计。通过编写简单的脚本，用户可以自定义快捷键（Hotkeys）、文本替换（Hotstrings）以及实现复杂的桌面自动化任务。

## 2. 核心功能

- **快捷键重映射**：修改键盘按键或组合键的功能。
- **自动更正与热字串**：输入缩写词自动替换为长段文字。
- **GUI 编程**：可以快速创建简单的桌面窗口和表单。
- **窗口管理**：自动移动、缩放或关闭特定的应用程序窗口。
